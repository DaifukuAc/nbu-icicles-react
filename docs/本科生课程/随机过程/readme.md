上的是2022秋JXJ的随机过程。课本是张波的应用随机过程。课上很多人在摸鱼，老师比较随意，也不挂人，都会捞的。

## 期末考试

期末考试的组成如下

- 叙述题 25分
- 填空题 15分
- 计算题 40分（5个）
- 证明题 20分（2个）

范围是1-9章，第四章更新过程不考。

以下是个人回忆的期末考题：

 - 条件期望的性质
 - possion过程
 - 首达概率
 - 正常返
 - 离散鞅
